Matt's LEARN Tools

LEARN Tools

A personal collection of browser-based tools for creating visual assets and course content for use in LEARN. No installation required — open any tool and get started.

Browser tools

These tools run entirely in your browser — nothing is uploaded to any server. Click to open.

Video

Video Thumbnail Creator

Create thumbnail images for videos on Media Hopper Create. Add the video title, presenter names, and course details, then export a PNG ready to use as the video's preview image in LEARN or elsewhere.

Course

Course Banner Image Creator

Generate themed banner images for LEARN course pages. Choose brand colours and add programme or course text to create a consistent visual identity that helps students recognise courses at a glance.

Open tool
Documents

LEARN Document Header Creator

Create branded 1920×320px header images for documents and pages within LEARN. Upload an icon, set your heading text, choose brand colours, and download a ready-to-use PNG.

Open tool
Modules

Learning Module Icon Creator

Design simple icon or photo based images for learning modules within LEARN courses. A consistent visual style across modules helps students navigate content more easily, even between different courses.

Open tool
Content

Quote Builder

Generate stylised pull-quote HTML to embed directly into LEARN pages. Choose from a selection of brand-coloured quote styles, enter your text, and copy the ready-to-paste HTML code.

Staff Contacts Page Builder

Create stylised HTML for course or programme contacts pages in LEARN, including staff photos, bios, and contact information. The app reads from a SharePoint list and generates ready-to-use HTML to paste into your course.

Access & setup: This app requires access to both the Power App and its associated SharePoint list. Staff photos also need to be uploaded to a content repository in a specific format. Click the link to request access through Microsoft, then get in touch and I can help with the setup.

Icons and Image Library
