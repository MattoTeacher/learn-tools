EMS Digital Education Unit

LEARN Tools

A collection of browser-based tools for creating branded visual assets for use in LEARN and across the Edinburgh Medical School's digital learning environment. No installation required — open any tool and get started.

Available tools

Click any tool to open it in your browser. All tools run locally — nothing is uploaded to any server.

Video

Video Thumbnail Creator

Create branded thumbnail images for videos hosted on Media Hopper Create — the university's video platform. Add the video title, presenter names, and course details, then export a PNG ready to attach as the video's preview image in LEARN or elsewhere.

Open tool
Course

Course Banner Image Creator

Generate themed banner images for LEARN course pages. Incorporate university brand colours and add programme or course text to create a consistent visual identity that helps students recognise and navigate between courses at a glance.

Open tool
Documents

LEARN Document Header Creator

Create branded 1920×320px header images for documents and pages within LEARN. Upload an icon, set your heading text, choose from university brand colours, and download a ready-to-use PNG — a faster, more consistent alternative to building headers in PowerPoint.

Open tool
Modules

Learning Module Icon Creator

Design simple icon-based images for learning modules within LEARN courses. Using university colours and a consistent visual style across modules helps students navigate content more easily, even when moving between different courses.

Open tool

Icon library

A shared library of flat icons is available on SharePoint for use with the tools above.

Icons and Image Library

A collection of icons, illustrations, and photos used in LEARN courses and the tools above. Hosted on SharePoint and accessible to EMS staff. Icons work best with the recolouring features in the tools — choose an icon, pick a university brand colour, and they'll match your document's visual theme.

Access note: This folder is on SharePoint and is available to EMS staff. If you do not have access, please contact Matt at the EMS Digital Education Unit: Med.digital.education@ed.ac.uk
